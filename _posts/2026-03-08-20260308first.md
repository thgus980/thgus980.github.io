## Playwright

### Playwright란?
- Playwright는 마이크로소프트(Microsoft)에서 개발한 오픈 소스 웹 자동화 및 테스팅 프레임워크. 우리가 마우스와 키보드로 브라우저에서 하는 모든 행동(클릭, 입력, 스크롤, 로그인 등)을 코드가 대신 수행하도록 만드는 것.
- codegen(코드 자동 생성) 기능이나 브라우저의 실행 과정을 녹화해서 보여주는 Trace Viewer 등이 기본으로 제공. (명령어: .venv\Scripts\python -m playwright codegen 웹사이트 주소 -> 이후 실제 브라우저 창과 Inspector 창이 뜸.)
- 사용 전 .venv\Scripts\playwright install chromium 명령어를 통해 Playwright에서 자체적으로 사용하는 브라우저 엔진을 설치해야 함.

### 응용: 모니터링 자동화(핵심 코드만)
```python
def crawl_work_report() -> str:
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        context = browser.new_context()
        page = context.new_page()
        try:
            # 1. Open login page
            log.info("Opening %s", TARGET_URL)
            page.goto(TARGET_URL, wait_until="networkidle", timeout=30_000)

            # 2. Open the login form (the site shows a "Log in" button first)
            log.info("Logging in as %s", LOGIN_ID)
            page.get_by_role("button", name="Log in").click()
            page.get_by_role("textbox", name="Account").fill(LOGIN_ID)
            page.get_by_role("textbox", name="Password").fill(LOGIN_PW)
            page.locator("form").get_by_role("button", name="Log in").click()
            page.wait_for_load_state("networkidle", timeout=30_000)
            log.info("Logged in – current URL: %s", page.url)

            # 3. Expand navigation sidebar, open 로그 parent menu, click 워크 리포트
            log.info("Navigating to 워크 리포트")
            page.get_by_role("navigation").get_by_role("button").click()
            page.get_by_role("link", name="로그", exact=True).click()

            pre_click_url = page.url
            page.get_by_role("link", name="워크 리포트").click()

            # SPA navigation: wait for the URL to change, then let content render
            try:
                page.wait_for_url(lambda url: url != pre_click_url, timeout=10_000)
            except PlaywrightTimeoutError:
                # URL did not change (hash/in-page routing) — wait for rendering
                page.wait_for_timeout(3_000)

            page.wait_for_load_state("networkidle", timeout=30_000)
            log.info("워크 리포트 loaded – URL: %s", page.url)

            # 4. Extract text content
            raw = page.inner_text("body")
            log.info("Extracted from <body> (%d chars)", len(raw))

            return clean_text(raw)

        except PlaywrightTimeoutError as exc:
            log.error("Timeout during crawl: %s", exc)
            raise
        finally:
            browser.close()

```
