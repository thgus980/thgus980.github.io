### LOB
- LOB: 대용량 데이터를 저장학 위한 데이터 타입, VARCHAR로는 담기 어려운 크기의 데이터를 저장할 때 사용
- CLOB: LOB 중에서도 문자 기반 대용량 데이터 타입, 긴 텍스트를 저장할 때 사용
- 오라클 DB에서는 긴 텍스트를 안정적으로 저장하기 위해 CLOB 타입을 사용
- 정리: CLOB는 긴 텍스트를 저장하는 대용량 문자 데이터 타입이며, FAQ 게시판처럼 본문이 길 수 있는 데이터를 저장할 때 사용
- CLOB가 일반 문자열과 다른 점
  1. 일반 문자열이 아니다: LOB 객체는 .strip() 같은 문자열 메서드를 바로 사용할 수 없음
  2. 읽기 과정이 필요하다: .read()를 호출해 문자열로 반환해야함
  3. 반면, VARCHAR2와 같은 일반 문자열 타입은 위와 같은 추가 처리가 필요 없음
